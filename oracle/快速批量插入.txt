方法2：
使用oracle自带的工具sqlldr
sqlldr用法示例：
1. 假设存在一张如下的表
create table user (
  id number,
  name varchar2(20)
)
2. 创建一个文本文件test.txt，用于存放需要插入上表的数据
cat test.txt
1,张三
2,李四
3,王五
3. 创建一个控制文件test.ctl，用于控制sqlldr的行为
cat test.ctl
load data
infile 'test.txt'
insert into table user
fields terminated by ","
(id,name)
4. 在oracle用户下执行sqlldr userid=userName/passwd@oracleIP:oraclePort/oracleSID control=test.ctl silent=header,feedback
如果有多个文件，可以多进程后台执行sqlldr
